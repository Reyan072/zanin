چالاکیێـــن مـــە



( چوون بۆ سینەمایێ )



ژ بۆ هندێ کۆ زمانێ فێرخازی باشتڕ لێ بهێت و پتر فێر ب بیتە زاراڤێن زمانی ئەم فێرخازان د بەینــە سینــەمایێ ژ بۆ تەماشەکرنا فلمان هەر وەسا بەڕی ئەم فێرخازێن خۆ ب بــەین ژ لایێ مامۆستایێن مەڤە د هــێتـە زانین کا ئەو فلمـــە چ جورە فلمــەکــە و چەند خولــەکـــە و کا چ رادە مفا تیدا هەیە







( سەیران )



مەرەم ژ چونا سەیرانا ئەوە فێرخاز خوشی و مفای ژ کۆرسێن مە ب بینیت و د هەر سەیرانەکێ دا چەندین پێزانینـێــن نی فیڕ ب بیت







( دانا سمینارا )



چـوونا کللـــینیکــا ، هەروەسا دانــا سمـینارا بۆ زاڕۆك و مەزنـــان یێن دەرونـی و کومـەلایـەتـی ژ بەرکو ساخلەمیا فێرخازی بۆ مە گەلەکا گرنگــــە ئەم ڤان سمینارا پیشــکێشی فێرخازێن خۆ دکـەین











( چوون بۆ باخـچـــێ گــیانـەوەڕان )



مـەرەم ژ چوونا باخـچـــێ گیانەوەران ئەوە پیچــەك ســەقایێ فێرخـــازی بهێتـــە گهــورین ، هەروەسا فێڕ ب بیتــە چەندین ناڤێن گیانەوەران و پێزانــین لـدۆڕ گیانــەوەری











زێدەباری چەندین جهــ و چالاکیێن دی ژ وانژی

( گێرانا ئاهەنگێن دەرچونێ و بیرەوەریێن کوردان )

( گەشتێن زانستی بۆ رێڤەبەریێن دهوکێ )
